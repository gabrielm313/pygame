Como rodar o jogo:
Antes de começar a jogar plugue os controle de Xbox no seu notebook. No arquivo astrored.py, inicie o arquivo e ele te enviará para uma tela de menu. Em que temos 3 opções: "Jogar", "Tutorial" e "Ranking". Para acessar o Tutorial, clique "B" no controle e passe a pagina com "A" e inicie o jogo. Logo irá aparecer para colocar o nome da dupla e assim o jogo inicia.
