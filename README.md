Como rodar o jogo:
Antes de começar a jogar plugue 2 controles de Xbox no seu notebook/PC. Para rodar o jogo, acesse o arquivo main.py e coloque para rodar. Vai aparecer uma interface para ser acessada pelo mouse, ou seja, o usuário vai ver "Jogar", "Tutorial" e "Ranking" e escolhe qual opção quer acessar. Em "Ranking" vai aparecer os players com os melhores tempos de jogo. Em "Tutorial", aparece os comandos utilizados no controle durante o jogo, para você paginar só clicar no espaço do seu teclado (são 2 páginas no total). Em "Jogar", vai rodar o jogo, primeiro vai aparecer para escrever os nomes do players (nome que vai aparecer no ranking) e depois começa uma interface de história que é passada pelo espaço do notebook. Após a história, aparece a primeira parte jogável, contra o primeiro Boss, quando o primeiro Boss for derrotado, aparece o cartaz para o próximo e começa a fase contra o 2° boss. E no final, é um duelo de reação do player 1 vs player 2, quem ganhar aparece no ranking.
A inspiração do jogo foi o jogo Sunset Riders, mas sem utilizar nenhum código deles, apenas sendo usado de inspiração para estilo de jogo, músicas de fundo e mecânica dos personagens.
O arquivo utiliza apenas a biblioteca Pygame.
O código é separado em 10 arquivos e rodado na pasta main.py, cada arquivo roda uma parte do jogo e carrega elementos como personagens, fases de boss, etc.
Com auxilio de inteligências artificiais generativas como chatgpt, gemini e cloude, foi possível o desenvolvimento e criação de funções e classes.
